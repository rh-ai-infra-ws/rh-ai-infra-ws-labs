# Openshift AI Infrastructure Workshop

This repository contains documentation and materials for the Openshift AI infrastructure workshop 🚀

## Workshop Focus

- Openshift AI platform overview
- Infrastructure components and configuration
- Accelerator configuration
- Deployment best practices
- Distributed Inference
- Model as a Service
- Monitoring AI workloads

## Docsify

Docsify is a documentation generator that renders markdown files directly in the browser.

- No build step required
- Write plain Markdown
- Supports sidebar, search, and custom themes
- Good for lightweight workshop docs

### Local preview

From the repository root:

```bash
cd docs
npx docsify-cli serve docs
```

### Usage

- **Home page:** this file (`README.md`).
- **Sidebar:** `docs/_sidebar.md`.
- **New pages:** add Markdown under `docs/` and link them from `_sidebar.md`.
